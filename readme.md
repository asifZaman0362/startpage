# Installation

Setting as startup homepage is rather trivial.
Just assign the file:///path/to/index.html as the homepage location

For using as a New Tab page, there are a few work arounds but most of them are kinda dodgy and
there's no guarantee that a particular method is going to keep working. The most reliable way to get
it to work is to start a local webserver with the required files. Then use the [New Tab Override extension](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/)
and set http://0.0.0.0 as the custom URL.
